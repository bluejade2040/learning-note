
# 新增、刪除、查詢、修改功能說明


```python
新增:
    程式碼 => def insert(self, root, val)
    用途為將值依照左小右大的規則新增至樹內
    參數root為樹頂、val為值
```


```python
    def insert(self, root, val):
        """
        :type root: TreeNode
        :type val: int
        :rtype: TreeNode(inserted node)
        """
        if root == None:         #如果tree是空的，
            return TreeNode(val) #回傳新節點
        else:
            if val<=root.val:  #將小於等於root的值移往左邊
                if root.left is None:
                    root.left = TreeNode(val)
                else:
                    self.insert(root.left,val)
            else: #大於往右邊
                if root.right is None:
                    root.right = TreeNode(val)
                else:
                    self.insert(root.right,val)
        return root
```


```python
刪除:
    程式碼 => def delete(self, root, target)
    將目標節點進行刪除，剩餘進行補足
    參數root為樹頂、target為輸入目標值
```


```python
    def delete(self, root, target):
        """
        :type root: TreeNode
        :type target: int
        :rtype: TreeNode(the root of new completed binary search tree) (cannot search())
        """
        if not root:
            return None
        
        else:
            if root.val > target: 
                root.left = self.delete(root.left, target)

            elif root.val < target: 
                root.right = self.delete(root.right, target)

            else: 
                if not root.right:
                    temp = root.right  
                    root = None 
                    return temp

                if not root.left:
                    temp = root.left  
                    root = None
                    return temp
         
                temp = self.minNode(root.right) 
                root.val = temp.val 
                root.right = self.delete(root.right, temp.val)
        
        return root
```


```python
查詢:
    程式碼 => def search(self, root, target)
    用途為查詢目標值所在位置，也有定位的作用
    參數root為樹頂、target為輸入目標值
```


```python
    def search(self, root, target):
        """
        :type root: TreeNode
        :type target: int
        :rtype: TreeNode(searched node)
        """
        if root == None : 
            return root  #root是None,回傳None
        if root.val == target:
            return root  #root是target回傳target
        if target >= root.val: 
            return self.search(root.right,target) 
        else:  
            return self.search(root.left,target)
        return root
```


```python
修改:
    程式碼 => def modify(self, root, target, new_val)
    將目標節點替換，刪除目標、加入新值
    參數root為樹頂、target為輸入目標值、new_val為新加入的值
```


```python
    def modify(self, root, target, new_val):
        """
        :type root: TreeNode
        :type target: int
        :type new_val: int
        :rtype:TreeNode(the root of new completed binary search tree) (cannot search())
        """
        while self.search(root, target):
            self.delete(root, target)
            self.insert(root, new_val)
            
        return root
```
